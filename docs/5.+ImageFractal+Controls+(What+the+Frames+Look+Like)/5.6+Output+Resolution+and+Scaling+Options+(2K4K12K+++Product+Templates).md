## 5.6 Output Resolution and Scaling Options (2K/4K/12K + Product Templates)

This section describes how to control the output frame resolution and how input images are scaled or cropped to fit those resolutions. You can choose one of three fixed “K” resolutions—2K, 4K or 12K—via the `global_output_scale_k` parameter, and you can fine-tune input image scaling/downscaling behavior or crop widescreen frames. Optionally, you can generate additional output variants sized to FineArtAmerica product templates.

### 5.6.1 Output Scale Presets

The global variable

```cpp
int global_output_scale_k = 12;
```

determines the target resolution for every output frame. Valid values and their associated dimensions are:

| `global_output_scale_k` | Resolution Identifier | Width × Height |
| --- | --- | --- |
| 2 | 2K | `global_nwidth_2k` × `global_nheight_2k` = 1920 × 1080 |
| 4 | 4K | `global_nwidth_4k` × `global_nheight_4k` = 3840 × 2160 |
| 12 | 12K | `global_nwidth_12k` × `global_nheight_12k` = 12000 × 12000 |


You can override this default by passing a command-line argument (argument index 39):

```cpp
if (nArgs > 39) {
    global_output_scale_k = atoi(szArgList[39]);
    // must be 2, 4 or 12; otherwise reverts to previous value
}
```

### 5.6.2 Input Image Scaling and Cropping Modes

Before fractal rendering, each input image is prepared according to several configurable modes:

1. **Upscaling Policy** (`global_input_upscale`)

Controls whether input images smaller than the target resolution are enlarged:

- 0 = never upscale
- 1 = allow slight upscaling (up to 25% area increase)
- 2 = allow any upscaling

```cpp
   int global_input_upscale = 0;  // default: no upscaling
```

1. **Downscaling/Cropping Policy** (`global_input_downscale`)

Controls how images larger than the target resolution are reduced:

- 0 = crop only (random sub-rectangle of target size)
- 1 = slight downscale or crop
- 2 = allow any downscale

```cpp
   int global_input_downscale = 0;  // default: crop only
```

1. **Uniform Input Sizing**

When you need all inputs to share the same dimensions before processing, set:

```cpp
   int global_INPUTIMAGES_ARE_TOBESCALEDTOFAAPRODUCTTEMPLATESIZE = 0;
   #define INPUTIMAGES_ARE_TOBESCALEDDOWNTOIDENTICALSIZE_WIDTH  -1
   #define INPUTIMAGES_ARE_TOBESCALEDDOWNTOIDENTICALSIZE_HEIGHT -1
```

A positive value for `global_INPUTIMAGES_ARE_TOBESCALEDTOFAAPRODUCTTEMPLATESIZE` will force every input to be uniformly rescaled down to the smallest width/height found (or the specified defines) .

1. **Widescreen Frame Cropping**

If your library contains 16:9 images with black bars, enable:

```cpp
   bool global_INPUTIMAGES_ARE_WIDESCREENFRAMES = false;
```

The code will then crop a centered 16:9 sub-rectangle—first taking 90% of the width and height, then centering—to remove letterboxing .

### 5.6.3 FineArtAmerica Product Template Sizes

When `global_OUTPUTIMAGES_ARE_TOBESCALEDTOFAAPRODUCTTEMPLATESIZE` is set to 1, after generating each fractal frame at the chosen “K” resolution, the program will produce additional variants scaled to a set of FineArtAmerica product dimensions. The following globals define the template catalog:

```cpp
#define GLOBAL_FAA_PRODUCT_TEMPLATE_NUMBER 20

int   global_faa_product_template_nwidth[]  = {
    12000, 3543,  560,  3400, 2700,
     8640, 7875, 1245, 1125, 1725,
     3600, 4464, 7104, 1756,  830,
     5100, 2325, 2925, 3825, 1800
};
int   global_faa_product_template_nheight[] = {
    12000, 4783, 1180, 6800, 1120,
     8640, 9375, 1170, 1900, 1050,
     2700, 4464, 7392, 2481,  830,
     6000, 2325, 2925, 2475, 5256
};
const char* global_name_faa_product_template[] = {
    "generic",      "apparel",        "batterycharger", "beachtowel",
    "coffeemug",    "duvetcover",     "fleeceblanket",  "masks",
    "phonecover",   "pouch",          "puzzle",         "roundtowel",
    "showercurtain","spiralnotebook", "stickers",       "tapestry",
    "throwpillow",  "totebag",        "weekenderbag",   "yogamat"
};
```

| Template Index | Name | Width (px) | Height (px) |

| 1 | apparel | 3543 | 4783 |
| 2 | batterycharger | 560 | 1180 |

| 4 | coffeemug | 2700 | 1120 |
| 5 | duvetcover | 8640 | 8640 |

| 10 | puzzle | 3600 | 2700 |

| 13 | spiralnotebook | 1756 | 2481 |

When enabled, the code rescales the greatest dimension of the fractal frame to match the template’s greater of width/height, then adjusts the other dimension to preserve aspect ratio before saving each variant. This allows you to produce a suite of ready-to-print images for FineArtAmerica products directly from your fractal animation pipeline.
