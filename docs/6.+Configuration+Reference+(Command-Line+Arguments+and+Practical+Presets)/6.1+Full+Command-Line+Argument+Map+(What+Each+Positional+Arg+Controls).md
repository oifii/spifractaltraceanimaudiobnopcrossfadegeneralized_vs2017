# 6. Configuration Reference (Command-Line Arguments and Practical Presets)

This section documents every positional command-line argument supported by the `spifractaltraceanimaudiobnopcrossfade` application. Each argument corresponds to a `global_…` variable initialized in the `_tmain` function of `spifractaltraceanimaudiobnopcrossfade_moving-frames.cpp` . Arguments beyond those listed are ignored.

### 6.1 Full Command-Line Argument Map (What Each Positional Arg Controls)

| Position | Variable | Description |
| --- | --- | --- |
| 1 | global_imagefolder | Path to the input image folder containing source frames. |
| 2 | global_outputimagefolder | Path where transformed frames will be written (must exist). |
| 3 | global_imageextension | File extension filter for input images (e.g. `.bmp`, `.png`). |
| 4 | global_maxnumberofimagestoload | Maximum number of images to load from the input folder (`-1` = all). |
| 5 | global_fractaltrace_outsidetype | How to handle samples outside image bounds (`0`=WRAP, `1`=BLACK, `2`=WHITE, `3`=WRAP_WITH_VARIABILITY_MATRIX). |
| 6 | global_fractaltrace_type | Fractal type (`0`=Mandelbrot, `1`=Julia). |
| 7 | global_fractaltrace_depth | Iteration depth for the fractal trace (e.g. `2` or `3`). |
| 8 | global_fractaltrace_bailout | Bail-out threshold (floating-point). |
| 9 | global_fractaltrace_jx | Julia set parameter Jₓ (only for `type=1`). |
| 10 | global_fractaltrace_jy | Julia set parameter J_y (only for `type=1`). |
| 11 | global_fractaltrace_xmin | Minimum X coordinate in fractal space. |
| 12 | global_fractaltrace_xmax | Maximum X coordinate in fractal space. |
| 13 | global_fractaltrace_ymin | Minimum Y coordinate in fractal space. |
| 14 | global_fractaltrace_ymax | Maximum Y coordinate in fractal space. |
| 15 | global_audiofilename | Path to the input audio file (for aubio analysis and optional ffmpeg merge). |
| 16 | global_maxnumberofframeperseedaudiofile | (Unused) maximum frames per audio file segment. |
| 17 | global_maxwindowwidthtranslationmultiple | Multiplier controlling translation speed of the fractal zoom window. |
| 18 | global_outputvideoframepersecond | Target frames-per-second for output video wrapping. |
| 19 | global_ffmpegpath | Path to `ffmpeg` executable for wrapping frames into MP4. |
| 20 | global_framefilenameprefix | Prefix for generated frame filenames (e.g. `frame_`). |
| 21 | global_videofilenameext | Extension for the final video file (e.g. `.mp4`). |
| 22 | global_aubiotrackpath | Path to `aubio` beat-tracking CLI. |
| 23 | global_mergeaudiowithfinalvideo | `0` or `1` – merge original audio into the final MP4. |
| 24 | global_fasttranslationspeed | `0` or `1` – use fast translation stepping (per-segment instead of per-frame). |
| 25 | global_zoomwindowsfilename | Path to a text file with per-beat “zoom window” definitions. |
| 26 | global_newimageinverseprobability | Integer `1–100` percent chance to invert image selection on change. |
| 27 | global_aubionotespath | Path to `aubio` pitch-note CLI. |
| 28 | global_aubionotesquantize | Quantization level for aubio-note segmentation (`0–5`). |
| 29 | global_defaultimagechangeperiod_sec | Default period in seconds between image changes (when no BNOP events). |
| 30 | global_aubiopitchpath | Path to `aubio` pitch-tracking CLI. |
| 31 | global_aubiopitchquantize_hz | Minimum pitch-change threshold in Hz. |
| 32 | global_aubioonsetpath | Path to `aubio` onset-detection CLI. |
| 33 | global_aubioonsetthreshold | Onset threshold (`0.001–0.900`). |
| 34 | global_aubioonsetminimumonsetinterval_sec | Minimum seconds between detected onsets (`≥0.020`). |
| 35 | global_bnop | BNOP mode string: `"beat"`, `"note"`, `"onset"`, or `"pitch"`. |
| 36 | global_imagemagickconvertpath | Path to ImageMagick `convert` (only for 32-bit builds; unused in 64-bit). |
| 37 | global_maxnumberofframeforcrossfades | Number of frames over which to cross-fade adjacent segments (`>2` to enable, `-1` = disabled). |
| 38 | global_newzoomwindowforeachframe | `0` or `1` – pick a new zoom window each frame instead of each beat. |
| 39 | global_output_scale_k | Output resolution scale: `2`=2K, `4`=4K, `12`=12K (12 000×12 000). |
| 40 | global_output_video | `"yes"` or `"no"` – also wrap frames into a final video file. |
| 41 | global_input_upscale | Input upscale policy (`0`=none, `1`=slight, `2`=aggressive). |
| 42 | global_input_downscale | Input downscale policy (`0`=none, `1`=slight, `2`=aggressive). |
| 43 | global_minnumberofbeats_beforechange | Minimum number of beats before allowing an image change (beat mode only). |


The application computes derived values such as `global_defaultimagechangeperiod_framenumber` and enforces valid ranges for thresholds and modes immediately after parsing. Any arguments beyond position 43 are ignored .
