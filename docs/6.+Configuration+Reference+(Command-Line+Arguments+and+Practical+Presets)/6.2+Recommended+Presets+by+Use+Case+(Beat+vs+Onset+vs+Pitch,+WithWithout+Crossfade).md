### 6.2 Recommended Presets by Use Case

This section collects practical, task‐oriented presets—each tied to the most relevant global variables—to help you get up and running quickly for common artistic goals.

#### 6.2.1 Beat-Driven Smooth Transitions

Use this preset when you want frame changes synced to musical beats, with gentle crossfades delivering a clip-like, cinematic feel.

| Global Variable | Recommended Value | Effect |
| --- | --- | --- |
| global_bnop | `"beat"` | Segment frames on detected beats |
| global_maxnumberofframeforcrossfades | `30` | Crossfade over 30 frames (~1 s at 30 fps) |
| global_fractaltrace_outsidetype | `0` | Wrap fractal edges around input image |


Command-line example (positional args 35–37):

```plaintext
... <audio_onset_path> <bnop=beat> <imagemagick_path> <maxcrossfadeframes=30> ...
```

---

#### 6.2.2 Onset-Based Fewer Cuts

Ideal for reducing the total number of edits by only switching frames on stronger onsets.

| Global Variable | Recommended Value | Effect |
| --- | --- | --- |
| global_bnop | `"onset"` | Segment frames on detected onsets |
| global_aubioonsetthreshold | `0.50` | Raise detection threshold for fewer onsets |
| global_maxnumberofframeforcrossfades | `5` | Short crossfades soften abrupt cuts (or set to `0` to disable) |
| global_fractaltrace_outsidetype | `0` | Wrap fractal edges |


---

#### 6.2.3 Pitch-Based Stable Sections

Perfect for long, stable visuals where only significant pitch shifts trigger a new frame.

| Global Variable | Recommended Value | Effect |
| --- | --- | --- |
| global_bnop | `"pitch"` | Segment frames on pitch‐change events |
| global_aubiopitchquantize_hz | `50.0` | Coarse quantization ignores minor pitch fluctuations |
| global_maxnumberofframeforcrossfades | `0` | Disable crossfade for crisp, immediate transitions |
| global_fractaltrace_outsidetype | `0` | Wrap fractal edges |


---

#### 6.2.4 Variability-Matrix Mode for Maximum Texture Variation

An experimental, “glitchy” mode that wraps the fractal trace across a matrix of input frames—ideal when each output frame should look drastically different.

| Global Variable | Recommended Value | Effect |
| --- | --- | --- |
| global_fractaltrace_outsidetype | `0` | Enable wrap mode so the variability‐matrix logic in DoFractalTrace can tap multiple input images |
| global_newzoomwindowforeachframe | `1` | Force a new zoom window on every frame, maximizing variation |
| global_maxnumberofframeforcrossfades | `-1` | Disable crossfade to preserve unpredictable, raw transitions |
