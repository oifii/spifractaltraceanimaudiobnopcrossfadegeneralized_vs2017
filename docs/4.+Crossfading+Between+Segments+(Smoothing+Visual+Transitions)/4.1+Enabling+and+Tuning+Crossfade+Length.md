### 4. Crossfading Between Segments (Smoothing Visual Transitions)

This feature inserts gradual crossfades between adjacent audio‐driven segments of generated frames, smoothing abrupt visual cuts and creating a more cohesive animation.

#### 4.1 Enabling and Tuning Crossfade Length

**User‐Facing Control**

- `**global_maxnumberofframeforcrossfades**`
- Global integer that caps the number of frames used for any single crossfade.
- Default is `-1` (no crossfades). To enable, this value must be set > 2.
- Supplied as the 38th command‐line argument:

```cpp
    if (nArgs > 37)
      global_maxnumberofframeforcrossfades = atoi(szArgList[37]); // must be > 2 to enable crossfade
```

**Activation Gate**

- Crossfading logic is only entered when:

```cpp
  if (global_maxnumberofframeforcrossfades > 2) {
      // iterate over audio‐based segment boundaries…
  }
```

**Crossfade Length Computation**

For each pair of consecutive segments (ending at `framenumber_start…framenumber_next` and beginning at `framenumber_next…framenumber_follow`):

1. Measure available frames on each side:

```cpp
   int maxPrev = framenumber_next – framenumber_start;
   int maxNext = *(iter+1) – framenumber_next;
```

1. Constrain to half of each segment (to avoid over-lapping beyond segment bounds):

```cpp
   int halfPrev = maxPrev / 2;
   int halfNext = maxNext / 2;
```

1. Apply the global cap and take the minimum:

```cpp
   int crossfadeFrames = min(halfPrev, halfNext);
   crossfadeFrames = min(global_maxnumberofframeforcrossfades, crossfadeFrames);
```

1. Round to the nearest even number for symmetrical blending:

```cpp
   crossfadeFrames = std::round(crossfadeFrames * 0.5f) * 2.0f;
```

**Final Gating Condition**

- To avoid imperceptibly short fades, a crossfade is actually performed only when:

```cpp
  if (crossfadeFrames > 3) {
      // perform crossfade blend over crossfadeFrames frames
  }
```

---

This tuning strategy ensures that crossfades never exceed the available frames in either segment, respect a user‐configurable maximum, and always use an even count of frames for balanced in-out blending.
