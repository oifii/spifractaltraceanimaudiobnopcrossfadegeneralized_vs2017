## 1.2 Building with Visual Studio 2017 (Win32 vs x64)

To build the `spifractaltrace` console application in Visual Studio 2017, open the `spifractaltrace.vcxproj` file at the root of the repository. You must choose both a Configuration (Debug or Release) and a Platform (Win32 or x64). The Win32 build links only FreeImage, FreeImagePlus and libsndfile, whereas the x64 build additionally links OpenCV (enabling the crossfade path).

### 1.2.1 Directory Layout & Dependencies

All third-party headers and libraries live under the `lib-src` folder (referenced via `..\lib-src\…` in the project). Ensure you have the following structure alongside your `.vcxproj`:

• lib-src\

– freeimage\Source

– freeimage\Wrapper\FreeImagePlus

– freeimage\Dist\FreeImage.lib

– freeimage\Wrapper\FreeImagePlus\Debug\FreeImagePlusd.lib

– freeimage\Wrapper\FreeImagePlus\Release\FreeImagePlus.lib

– freeimage_vs2017(x64)\Source

– freeimage_vs2017(x64)\Wrapper\FreeImagePlus

– freeimage_vs2017(x64)\Dist\x64\FreeImage.lib

– freeimage_vs2017(x64)\Wrapper\FreeImagePlus\dist\x64\FreeImagePlus.lib

– libsndfile\include

– libsndfile\lib\libsndfile-1.lib

– libsndfile(x64)\include

– libsndfile(x64)\lib\libsndfile-1.lib

– opencv\build\include

– opencv\build\x64\vc15\lib\opencv_world342.lib

– opencv\build\x64\vc15\lib\opencv_world342d.lib

This layout matches the relative paths in the project’s `<AdditionalIncludeDirectories>` and `<AdditionalDependencies>` settings.

### 1.2.3 Win32 vs x64: Key Differences

- **Win32 builds**

– Use 32-bit FreeImage/FreeImagePlus under `lib-src\freeimage`

– Do **not** link any OpenCV libraries (crossfade code is disabled)

– `AdditionalIncludeDirectories` points at the OpenCV headers, but no corresponding `.lib` is listed .

- **x64 builds**

– Use 64-bit FreeImage/FreeImagePlus under `lib-src\freeimage_vs2017(x64)`

– Link against `opencv_world342d.lib` (Debug) or `opencv_world342.lib` (Release), enabling the OpenCV-based crossfade path

– Source code includes OpenCV headers conditionally under `_M_X64` (e.g. `#if defined _M_X64` in `spifractaltrace.cpp`) .

### 1.2.4 First Successful Build & Run

1. Verify that all `lib-src` dependency folders match the structure above.
2. Open **spifractaltrace.vcxproj** in Visual Studio 2017.
3. Select **Solution Platforms** → **Win32** or **x64**, and **Debug** or **Release**.
4. Build the solution (`Build → Build Solution`).
5. Run the resulting executable from `.\Debug\Win32`, `.\Debug\x64`, `.\Release\Win32`, or `.\Release\x64`.

– On **x64**, the OpenCV-based crossfade feature will be available; on **Win32**, it will be skipped.
