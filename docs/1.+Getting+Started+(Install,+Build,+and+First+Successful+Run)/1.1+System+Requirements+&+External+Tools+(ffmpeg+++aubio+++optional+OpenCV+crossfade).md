# 1. Getting Started (Install, Build, and First Successful Run)

## 1.1 System Requirements & External Tools

Before building and running the `spifractaltraceanimaudiobnopcrossfade` console application, ensure the following prerequisites and external tools are installed and configured:

### 1.1.1 Supported Platform & Build Environment

- Microsoft Windows (tested on Windows 10).
- Visual Studio 2017 with C++ workload.
- Win32 (x86) and x64 configurations available in the provided `.vcxproj`.
- The x64 build links against OpenCV for crossfade support; the x86 build uses ImageMagick’s `convert`.

### 1.1.2 Core Libraries

- FreeImage (included via `#include "FreeImage.h"`).
- libsndfile (SFML): `#include "sndfile.h"` for audio file I/O.

### 1.1.3 FFmpeg (Required)

Used for:

1. Converting any audio input to a mono WAV for Aubio analysis.
2. Wrapping generated frames into an H.264-encoded MP4, optionally merging the original audio.

Default path (can be overridden via command-line):

```cpp
string global_ffmpegpath = "D:\\spibin\\ffmpeg\\ffmpeg";  
```

Override argument index 19:

```cpp
if(nArgs > 19)
    global_ffmpegpath = szArgList[19];
```

Sample mono-conversion command in code:

```cpp
string mono_audiofilename = "mono_audiofilename.wav";
systemcommand = "\"" + global_ffmpegpath + "\" -y -i \"" + global_audiofilename + "\" -ac 1 " + mono_audiofilename;
system(systemcommand.c_str());
```

### 1.1.4 Aubio CLI Tools (Required)

Aubio extracts beat, note, onset, and pitch events from the mono WAV. Four separate executables are invoked:

| Tool | Default Path | Override Arg Index |
| --- | --- | --- |
| aubiotrack | `D:\spibin\aubio\aubio-0.4.6-win32\bin\aubiotrack` | 22 |
| aubionotes | `D:\spibin\aubio\aubio-0.4.6-win32\bin\aubionotes` | 27 |
| aubiopitch | `D:\spibin\aubio\aubio-0.4.6-win32\bin\aubiopitch` | 30 |
| aubioonset | `D:\spibin\aubio\aubio-0.4.6-win32\bin\aubioonset` | 32 |


```cpp
if(nArgs > 22) global_aubiotrackpath      = szArgList[22];
if(nArgs > 27) global_aubionotespath      = szArgList[27];
if(nArgs > 30) global_aubiopitchpath      = szArgList[30];
if(nArgs > 32) global_aubioonsetpath      = szArgList[32];
```

Each tool writes event times to a text file (e.g. `aubiotrack_beattimes.txt`), which the application then parses into frame-number triggers.

### 1.1.5 Optional Crossfade Support

#### 32-bit (x86) Builds: ImageMagick `convert`

When compiled for Win32, frame crossfades use ImageMagick’s CLI:

- Default path:

```cpp
  string global_imagemagickconvertpath = "C:\\Program Files (x86)\\ImageMagick-7.0.8-Q8\\convert";
```

- Override argument index 36:

```cpp
  if(nArgs > 36) global_imagemagickconvertpath = szArgList[36];
```

- Maximum frames per crossfade (arg 37) must be > 2 to enable crossfade; otherwise disabled:

```cpp
  if(nArgs > 37) global_maxnumberofframeforcrossfades = atoi(szArgList[37]);
```

Crossfade loop (blend two boundary frames):

```cpp
systemcommand = "\"" + global_imagemagickconvertpath + "\" "
              + "\"" + frameA + "\" \"" + frameB + "\" "
              + "-delay 20 -morph " + to_string(nMorph) + " "
              + outputPattern;
system(systemcommand.c_str());
```

#### 64-bit (x64) Builds: OpenCV

The x64 project links against OpenCV (`#include "opencv2/imgcodecs.hpp"`) and uses `cv::addWeighted` for crossfades:

```cpp
cv::Mat src1 = cv::imread(frameA);
cv::Mat src2 = cv::imread(frameB);
cv::Mat dst;
cv::addWeighted(src1, alpha1, src2, alpha2, 0.0, dst);
cv::imwrite(outputFrame, dst);
```

OpenCV support is enabled under `_M_X64` in the source and the VS project is configured to link the appropriate OpenCV libraries.

### 1.1.6 Summary of Configurable Paths & Arguments

| Argument Index | Global Variable | Purpose |
| --- | --- | --- |
| ---------------: | ----------------------------------------- | --------------------------------------------------------------- |
| 19 | `global_ffmpegpath` | FFmpeg binary |
| 22 | `global_aubiotrackpath` | Aubio beat tracker |
| 23 | `global_mergeaudiowithfinalvideo` | Toggle audio merge (0/1) |
| 27 | `global_aubionotespath` | Aubio note extractor |
| 28 | `global_aubionotesquantize` | Note quantization level |
| 30 | `global_aubiopitchpath` | Aubio pitch detector |
| 31 | `global_aubiopitchquantize_hz` | Pitch quantization (Hz) |
| 32 | `global_aubioonsetpath` | Aubio onset detector |
| 33 | `global_aubioonsetthreshold` | Onset detection threshold |
| 34 | `global_aubioonsetminimumonsetinterval_sec` | Min inter-onset interval (s) |
| 35 | `global_bnop` | Event type: “beat”/“note”/“onset”/“pitch” |
| 36 | `global_imagemagickconvertpath` | ImageMagick `convert` (x86 crossfade) |
| 37 | `global_maxnumberofframeforcrossfades` | Max frames to blend per segment |


All paths and thresholds have sane defaults but can be customized at runtime by supplying additional command-line arguments  .
